# 浏览器插件:标签页管理器

一个很简洁的浏览器插件，用于汇总和管理当前窗口中打开的所有标签页,主要是针对一个窗口打开N个标签页的场景。

## 功能特点

- **按域名分组**：自动将标签页按网站域名分组展示，不同网站的标签页归为不同区块
- **颜色区分**：每个域名区块使用不同的颜色区分，颜色基于域名生成
- **网站logo**：显示每个标签页的网站favicon图标
- **页面标题**：显示标签页的标题，最多展示两行，多则自动折叠
- **重复标签页检测**：检测并提示重复的标签页，支持一键关闭重复页面
- **标签页管理**：支持点击关闭标签页，点击标签页标题切换到对应页面
- **撤销关闭**：关闭标签页后会显示toast提示，可点击"撤销"恢复最近关闭的标签页

## 核心功能实现

### 1. 标签页数据获取与域名分组
- 使用Chrome的`tabs API`获取当前窗口的所有标签页
- 通过解析URL提取域名，将同一域名的标签页分组
- 基于域名生成唯一的颜色，确保不同域名使用不同颜色

### 2. 界面渲染
- 按域名分组展示标签页
- 每个标签页显示网站favicon和标题
- 标题最多显示两行，超过部分自动折叠

### 3. 交互功能
- **点击标签页**：切换到对应标签页
- **点击关闭按钮**：关闭标签页并显示撤销提示
- **重复标签页检测**：提示用户并提供关闭选项
- **撤销关闭**：恢复最近关闭的标签页

### 4. 技术实现
- 使用原生JavaScript和Chrome扩展API
- 响应式设计，适配不同屏幕尺寸
- 模块化代码结构，易于维护和扩展

## 安装方法

1. 打开Chrome浏览器，进入扩展程序管理页面（chrome://extensions/）
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择插件目录
5. 插件将被安装到浏览器中，点击插件图标即可使用

## 使用说明

1. **查看标签页**：点击浏览器工具栏中的插件图标，查看当前窗口的所有标签页
2. **切换标签页**：点击标签页列表中的任意标签页，即可切换到对应页面
3. **关闭标签页**：点击标签页右侧的"×"按钮，关闭对应标签页
4. **撤销关闭**：关闭标签页后，点击toast提示中的"撤销"，恢复最近关闭的标签页
5. **处理重复标签页**：当插件检测到重复标签页时，会显示提示，可选择关闭所有重复页面

## 技术栈

- **前端**：HTML5, CSS3, JavaScript
- **浏览器API**：Chrome Extensions API (tabs, sessions)
- **构建工具**：无（原生实现）
